# itc_assignment_website_task1

This project is a multi-page static website created as part of an ITC assignment. It showcases different sections of a website such as About, Events, Team, Contact, and more using HTML, CSS, and JavaScript.

---

 Features

*  Multiple web pages:

  * Home (`index`)
  * About ITC (`about_itc`)
   - Club (`club`)
   - Communication (`comm`)
   - Tech teams(`tech_team`)
  * Events (`events`)
  * Team (`team`)
  * Contact (`contact`)
* Styled using external CSS
* Image assets included
* Basic interactivity using JavaScript (`event.js`)
* Simple and clean layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 💻 How to Run the Project

1. Download the project (or clone it from GitHub)
2. Extract the folder (if downloaded as ZIP)
3. Open the folder
4. Double-click `index.html`
5.The website will open in your browser.
